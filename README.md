# Aplicação DEVOPS

Este é um projeto básico de uma aplicação Node.js que utiliza o framework Express.js, tendo como intuito replicar conhecimentos adquiridos nas aulas de ***ferramentas de implantação contínua devops*** dando um foco maior na prática com base no conteúdo sobre sistemas de versionamento de código e ramificação.

## Como Usar

1. Certifique-se de ter o Node.js instalado em sua máquina.
2. Clone este repositório em seu computador.
3. Abra o terminal e navegue até o diretório do projeto.
4. Execute o seguinte comando para instalar as dependências: **npm install**
5. Após a instalação das dependências, execute o seguinte comando para iniciar o aplicativo: **npm start**
6. Para visualizar a página principal do site, abra o seu navegador e digite na barra de pesquisa : **localhost:3000**
7. Para visualizar os integrantes da aplicação, abra o seu navegador e digite na barra de pesquisa : **localhost:3000/integrantes**
